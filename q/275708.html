Method

We recursively build all patterns up to \$N\$, using 'a' and 'b'.

When the \$n\$-th pattern is reached, we convert it to upper case and save it in \$p\$. All subsequent regular occurrences of \$p\$ will appear in upper case as well.

At the end of the process:

  • If \$p\$ appears at offset \$o\$ in upper case, this is a regular match.
  • If \$p\$ appears at offset \$o\$ in lower or mixed case, this is a strange match.
  • If \$p\$ doesn't appear at all at offset \$o\$, this is obviously wrong.
