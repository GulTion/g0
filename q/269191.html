Hardcodes the matches into a dictionary (can be improved).

P actually represents the following dictionary:

P = {
    0b111111000110011000111111: 50000, # Border
    0b100010101000000101010001: 10000, # Cross
    0b000000111001100111000000: 1000, # Picture
    0b000010001000000100010000: 500, # Forward diagonal
    0b100000100000000001000001: 200, # Backward diagonal
    0b000010000100010000100001: 100, # Any vertical line
    0b000100001000100001000010: 100,
    0b001000010000000010000100: 100,
    0b010000100001000100001000: 100,
    0b100001000010001000010000: 100,
    0b000000000000000000011111: 50, # Any horizontal line
    0b000000000000001111100000: 50,
    0b000000000011110000000000: 50,
    0b000001111100000000000000: 50,
    0b111110000000000000000000: 50,
    0b100010000000000000010001: 30, # All corners
}

where each key is a mask to the actual grid. The 5 least-significant bits are the first row, the following 5 are the second row, and so on.

The dictionary is ordered by score, so the first entry matched will always evaluate to the highest score possible.
