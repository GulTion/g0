])~{1$}4*<@@)>+@@[=]+~

explanation:

])     # Separate the stack and the argument
~      # NOT the argument
{1$}4* # Duplicate the stack and argument so there are three of each, alternating
<      # Objects under the index
@@)>+  # Add the objects above the index
@@[=]+ # Add the object at the index
~      # Unpack everything
