Returns the exponent of the largest prime factor of \$n\$.

Proof of correctness

Not fully formal, but let's bound the number of values up to \$n\$ which return \$a\$. There's a trivial \$\Omega_a(\frac{n^{\frac{1}{\alpha}}}{\log(n)})\$ lower bound by looking at prime powers. For an upper bound, it's obviously \$\leq \sum_{p \leq n^\frac1{a}} \Psi(\frac{n}{p^a}, p)\$. From a theorem mentioned in the Integers without large prime factors, we have \$\Psi(\frac{n}{p^a}, p) = O_a (\frac{n}{p^a} e^{-\frac{\log(n)}{2 \log(p)}})\$, so it's bounded by the sum $$O_a(n \sum_{m=a}^{\lfloor \log_2(n) \rfloor} e^{-\frac m2} \sum_{n^\frac1{m+1} < k \leq n^{\frac1m}} \frac1{k^a}) = O_a (n \sum_{m=a}^{\lfloor \log_2(n) \rfloor} e^{-\frac m2 + \log(n)\frac{1-a}{m+1}})$$ Which we can show is \$O_a(n^\frac1{a} \log(n))\$, although I don't have time to do this in detail right now.
