A pair of monadic links that takes a list of lists and returns a list of polynomial coefficients.

The argument holding the input is rotated both horizontally and vertically by 1 such that the 0th row and column appear at the end. It is also transposed versus the ones shown in the question. This is done to best fit with Jelly’s 1-indexing and the fact that zero refers to the last row/column. The output starts with the zeroth coefficient and works upwards.

This is a direct implementation of the algorithm described in the Wikipedia article. I’m sure it can be be significantly shortened, but at least it’s working for now!

E=enumerate
def M(p,*P):
 for i in P:
  D={}
  for a,b in i.items():
   for A,B in p.items():D[a+A]=D.get(a+A,0)+b*B
  p=D
 return p
def O(p,*P):
 for i in P:
  for j,k in i.items():p[j]=p.get(j,0)+k
 return p

 L=[[(a,[i])for i,a in E(l)]]
