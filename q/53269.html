The correct statement among the options provided to convert a string representation of a number to an int[] containing each digit in C# is:

number.Select(i => i - '0').ToArray();

This statement utilizes the Select method to transform each character in the string into an integer by subtracting the ASCII value of '0', and then the ToArray method is used to convert the result into an integer array.
