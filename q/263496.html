b checks if a the expression encoded by the given number is balanced, by counting the difference of 1 digits minus 3 digits and returning 0 if the intermediate count goes below zero, or if the final count is not exactly zero.

f checks for all binary numbers k (up to l+r) if swapping the bits of the two inputs at the positions that are one in k will result in a pair of balanced strings.
