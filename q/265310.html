Python 3, 49 bytes

n=1
while 1/n:10**len({*str(n)})>n!=print(n);n+=1

Try it online!

Uses my 10** trick from 2014. Converting the string representation of \$n\$ to a set removes duplicates, and we check whether this decreases the length by comparing to \$10^n\$, which is bigger than all \$n\$-digit number but no \$(n+1)\$-digit numbers.

The 1/n is used to theoretically halt for large enough n because doubles become zero around 1e-308 due to finite precision. If the program may just continue to run forever after printing all the numbers, this can be:
