since (10^-1)^n = 10^-n we simply reverse the string, taking care of some special cases

  • g(x:y)z=g y$(x:z) and g[]z=z is reverse (where z is the accumulator)
  • g(x:'.':y)z=g y$x:'.':z swaps the position of the decimal point while reversing (in facts the unit remains the unit since 10^1 = 0.1^1 = 1)
  • f x=g x"0" calls our special reverse using an initial accumulator of ['0']. This initial accumulator adds a trailing zero to the result, which ensures there is at least one digit after the decimal point
  • with the guard elem '.'x we can add the decimal point to the input if it is missing
