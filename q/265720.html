What exactly

The set has 28 "generators" and 28 "terminators" (excluding the special case "22")

Starting from the generators, output the sequence until we find a terminator.

The interesting part is that the terminators can be found easily by looking at some string patterns. For example, none of the elements of the set has the string "2123" in it, but 17 terminators contain it.
