The built-in O (\setPartition) uses the following algorithm, which ensures that the first solution is exactly the one we want:

# a non-deterministic function that has multiple possible return values
def setPartition(A):
    if A is empty:
        return []
    else:
        x = A[0]
        B = A[1:]
        p = setPartition(B)
        if p is empty:
            return [[x]]
        else:
            non-deterministically choose an index i in range(len(p))
            # we will first try i = 0, then i = 1, and so on
            prepend x to p[i]
            return p
