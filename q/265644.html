This answer is a family of programs defined for any positive integer n, with the score approaching zero as n goes to infinity. Example for n=10:

print('b'*(23+len('aaaaaaaaaa')))

To construct the program for arbitrary values of n, simply repeat the 'a' n times instead.

The program will output 23+n 'b's followed by a newline. If any of the 'a's in the source code are removed, the output will have one fewer 'b'. If another character is removed, something else happens (either a syntax error, output of the empty string, output of 2+n 'a's, or output of 3+n 'a's).

This idea can likely be adapted to any language with string literals.
