Knight positions

Given \$j=\lfloor n/96\rfloor \in[0\dots9]\$, the 0-indexed position of the first knight is:

$$[0,0,0,0,1,1,1,2,2,3][j]$$

which can be obtained with:

~-j * 3 / 8 | 0

The 0-indexed position of the second knight is:

$$[1,2,3,4,2,3,4,3,4,4][j]$$

which can be obtained with:

~-j % 3 + 2 + j % 9 / 7 | 0
