Instead of accept reduced form p/q, this function may accept any even number q as well. We can extends the definition about symmetry as padding 0s at beginning do not affect the number value.

First, if q == 1, the output should be true iff p == 0.

Otherwise, we recursively test if bits read from right is matching bits read from left: We simply check if the last digits (p%2) is matching to the first digits (p&q*q/2). If they are matching, clear them to 0 and divide p by 2. So we can test remaining bits. If they do not match, set the left side bit to 1 on p and also divide p by 2. So after iteration, p must left to be non-zero.
