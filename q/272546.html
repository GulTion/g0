Outputs an infinite sequence. Same sequence as probably all the other answers.

Such sequences are sets of positive natural numbers \$2^i3^jn\$ such that \$2,3\not\mid n\$ and \$i-j\equiv c_n\pmod3\$ for some \$c_n\in\{0,1,2\}\$. We can fully define such a set by choosing \$c_n\$ for each \$n\$. Equivalently, choose one of \$n,2n,3n\$ to include in the set.

Therefore:

  • The set of such sets has cardinality \$3^{\{n\in\mathbb N^+|2,3\not\mid n\}}=2^{\aleph_0}\$.
  • Select the set \$S\$ defined by \$c_n=0\$, i.e. \$2,3\not\mid n\implies n\in S\$. The density of such \$n\$ is clearly \$\frac12\cdot\frac23=\frac13\$. However, we also have \$x\in S\implies 6x,8x,27x\in S\$. The density of \$S\$ is thus $$\frac13\left(1+\frac16+\frac1{6^2}\right)\left(1+\frac18+\frac1{8^2}+\cdots\right)\left(1+\frac1{27}+\frac1{27^2}+\cdots\right)= \frac{43}{91}\approx0.47253.$$
