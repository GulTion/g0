ṣ”#Ṗċ€”;%127p”Ọ

A full program printing equivalent Jelly code (as a monadic link it returns a list of lists of mixed types).

The first example is at Try it online! which yields this program.

How?

Counts up the ;s in each run between #s takes each modulo 127 and appends a cast to ordinal instruction, the monadic Ọ atom, after each.

Jelly implicitly pipes each value to STDOUT as it runs through a program like that i.e. 72Ọ101Ọ108Ọ108Ọ111Ọ44Ọ32Ọ119Ọ111Ọ114Ọ108Ọ100Ọ33Ọ would print Hello, world!.

ṣ               - split the result at #s
   Ṗ            - pop (remove the last result, list of trailing non-# characters)
      ”;        - literal ';'
    ċ€          - count for €ach
        %127    - modulo 127 (vectorises)
             ”Ọ - literal 'Ọ' (Jelly's cast to ordinal monadic atom)
            p   - Cartesian product - making a list of lists like [[72,'Ọ'],[101,'Ọ'],...]
                - implicit print (this smashes, printing something like: 72Ọ101Ọ...)

A note regarding input: Jelly can take string input in Python format (it will first attempt to evaluate input as Python code). The empty program may be input as "", and the hash-only programs as "#", "##", etc. Escape sequences are required for input containing backslashes and quotes.

        // nasm syntax with C comments because SE sucks

        // Because I use int 29h, you can't redirect the output with >.
        // Defining REDIRECTABLE will swap out the int 29h with an equivalent
        // int 21h:02h which can be redirected.
        // Unscored.
%ifdef REDIRECTABLE
%macro INT_29H
        push    ax
        push    dx
        mov     dl, al
        mov     ah, 0x02
        int     0x21
        pop     dx
        pop     ax
%endm
%else
%macro INT_29H
        int     0x29
%endm
%endif

        section .text
        [org 0x100]
        global start
start:
.reset:
        // Reset the accumulator
        xor     bx, bx
        // Main loop
.loop:
        // int 29h:08h: read char to AL (no echo)
        mov     ah, 0x08
        int     0x21
        // check for \r, if so, quit.
        cmp     al, 13
        je      .finalize
        // Check for semicolon
        cmp     al, ';'
        jne     .next
.semicolon:
        // Increment accumulator
        inc     bx
        // If we reach 127, reset the accumulator to 0.
        // This is the same as modulo 127 and
        // doesn't require an AAM or DIV setup.
        cmp     bl, 127
        je      .reset
.next:
        // Check for '#', if not, loop again
        cmp     al, '#'
        jne     .loop
.hash:
        // '#': print out code
        // mov al, <hardcoded acc>
        mov     al, 0xB0
        INT_29H
        mov     al, bl
        INT_29H
        // int 0x29
        mov     al, 0xCD
        INT_29H
        mov     al, 0x29
        INT_29H
        // Now reset the accumulator to 0
        jmp     .reset
.finalize:
        // Output ret instruction
        mov     al, 0xC3
        INT_29H
        // And exit
        ret

Note that the , is used to signify a list (versus atom) as "\"" is a list of one element, rather than being an atom.

Explanation:

Take input string, remove anything that isnt a # or a ;, split into lists on #, count number of elements in each list, perform mod 127 on the result and cast to a string:

{-1_"c"$mod[;127]count each"#"vs x inter"#;"} / ungolfed
{                                           } / lambda function
