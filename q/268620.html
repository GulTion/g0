s=>[...new Set(s.replace("B",""))].sort().join``.trim()

Try it online!

Method

Given the input string, e.g. "ABCWY ACWY B":

  • We remove the first "B" → "ACWY ACWY B"
  • We turn the string into a set → Set { 'A','C','W','Y',' ','B' }
  • We turn the set into an array → [ 'A','C','W','Y',' ','B' ]
  • We sort in lexicographical order → [ ' ','A','B','C','W','Y' ]
  • We turn it back into a string → " ABCWY"
  • We remove the leading space, if any → "ABCWY"
