How it works?

F groups \$4\$ steps: 3, 4, 5 and 6. Therefore, \$0 \leq j \lt 4\$.

Note that for steps 3, 4 and 5, a single letter is replaced by another single letter. This allows us to do

[c,'Dyk '[j]][c=='dac '[j]]

Also note that for step 6, ASCII character A is 65248 characters apart from ａ:

chr(ord(c)+65248)

Since they're different types of replacement, we can filter it using

[...,...][j>2]

where the element at index 1 will be returned for \$j=3\$ (step 6).
