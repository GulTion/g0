J, 33 bytes

-:&'u'{'117{a."_',:~'u:@',3":@u:]

Try it online!

A function which outputs a function. In J no-argument functions don't exist, so we call the fn with a dummy argument 0 which is ignored.

Probably a better approach out there, but this idea is:

  • Create a function which uses "Unicode" u: to translate the code points of the original input bad to unicode.
  • Everything is non-letter except for u, so that's the one input we need to defend against. So for that specfic case we return a function which returns the 117th letter of the J alphabet a., which is also u but whose code does not contain u: 117{a."_.

Google Sheets, 37 bytes

="="""&left(A1)&"""&"""&mid(A1,2,9^9)

Put the input in cell A1 and the formula in B1.

Given rin, the formula outputs ="r"&"in which is a valid formula in Google Sheets.

In Microsoft Excel, the formula requires a closing quote. It can be added by appending &"""".
