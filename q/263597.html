How?

The idea is to recursively build up the hyperfactorial using f. Each time, it checks if n is not 0. If that's true, it multiplies k (initially 1) by n**n and passes that, along with n-1 back to f. When n reaches 0, k is the hyperfactorial, and we are now ready to count trailing zeros.

The number of trailing zeros is limited by the amount of fives in the prime factorisation. If we recursively divide by 5, we can find the result. In g, we check if it's a multiple of 5, and if it is, we divide by 5 and pass that back to g, returning the result plus one. If it isn't, we return False (0).
