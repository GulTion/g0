This prints +-<>. The ASCII value for - (45) is two more than + (42), and the ASCII value for > (62) is two more than < (60). This observation did shorten the code, but I since the pattern was so simple, I couldn’t find a way for extracting it to be shorter than just including the code twice.

Explanation of the GolfScript:

# Stack starts with empty input
43    # "" 43
.     # "" 43 43
2+    # "" 43 45
60.2+ # "" 43 45 60 62
]     # ["" 43 45 60 62]
(     # [43 45 60 62] ""
:n    # Assign variable n to empty string because
      # n defaults to a newline that is printed when program ends
+     # "+-<>"

The new banned characters are +>.-<].
